# blackjack-py
Command line blackjack written in Python 3.

## Clone
    git clone https://gitlab.com/gdonald/blackjack-py.git

## Run
    cd blackjack-py
    ./blackjack.py

## Terminal

Increase your terminal font size to see the cards better:
